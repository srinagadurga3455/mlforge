# Changelog

## v1.0.0 — 2026
Initial release.

### What's included
- 11 data loaders (CSV, Kaggle, HuggingFace, PostgreSQL, MongoDB, API, S3, and more)
- Data ingestion (batch + streaming)
- Data validation (schema, missing values, quality checks)
- Data preprocessing (cleaning, type fixing, outlier handling, normalisation, text cleaning)
- Feature engineering (encoding, scaling, feature building, feature selection)
- 12 models, each in its own file (Random Forest, XGBoost, LightGBM, SVM, KNN, and more)
- Model training with auto train/test split and cross validation
- Hyperparameter tuning (grid search + random search)
- Model registry (save, version, compare, promote)
- Evaluation (R², MAE, RMSE, accuracy, F1, ROC-AUC, confusion matrix)
- Monitoring (drift detection, prediction logging)
